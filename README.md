# Counter App

A simple and elegant counter application built with Android Studio using Kotlin and Gradle. This app allows users to increment a counter value by clicking the "Click Me" button.

## Features

- 🔢 Simple counter that increments on button click
- 🎨 Clean and intuitive user interface
- 📱 Native Android application
- ⚡ Fast and responsive performance

## Technology Stack

- **Language**: Kotlin
- **IDE**: Android Studio
- **Build System**: Gradle
- **Platform**: Android

## Installation

### Prerequisites
- Android Studio (latest version recommended)
- Android SDK
- Kotlin plugin for Android Studio
- Git

### Steps
1. Clone this repository:
   ```bash
   git clone https://github.com/yash-144/Counter-app.git
   ```

2. Open Android Studio

3. Click on "Open an existing Android Studio project"

4. Navigate to the cloned repository folder and select it

5. Wait for Gradle sync to complete

6. Run the app on an emulator or physical device

## Usage

1. Launch the Counter App on your Android device
2. You'll see a counter display showing the current count (starts at 0)
3. Tap the "Click Me" button to increment the counter
4. Watch as the counter value increases with each tap!
